## Installation
To build the program code using **makefile**:

1. Clone the repository:
```bash
$ git clone git@github.com:ricardoyang00/univerter.git
$ cd univerter/
```

2. Build the program using make:
```bash
$ make
```

2. Run the program:
```bash
$ ./converter
```
